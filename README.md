

# RI1 – Projeto Vingadores

##  Disciplina
POO - Programação Orientada à Objetos

##  Curso
Análise e Desenvolvimento de Sistemas – Fatec

##  Descrição do Projeto
Esta atividade foi desenvolvido em JavaScript com o objetivo de criar objetos que representam personagens do universo Marvel (Vingadores) e seu principal rival, Thanos.

Todos os personagens foram armazenados em um Array e comparados entre si utilizando estruturas de repetição (for), conforme solicitado na atividade.

O sistema realiza:
- Criação de objetos com atributos numéricos
- Armazenamento em Array
- Comparação entre personagens
- Impressão dos resultados no console
- Indicação de qual personagem vence em cada atributo

##  Tecnologias utilizadas
- JavaScript

## ▶️ Como executar
